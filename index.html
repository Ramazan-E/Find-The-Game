Game Rules

1. The aim of the game is find the number, which is between 1000 and 9999. And, each digit of the number is different.
2. If you can find a digit from the number, but in the different place, you get -1, -2, -3 or -4.
For Example, If the number is 1234 and you input 4321, you get -4 or you input 2156, you get -2.
3. If you can find a digit from the number, but in the same place, you get +1, +2, +3 or +4.
For Example, If the number is 1234 and you input 1256, you get +2.
4. Good Luck!
